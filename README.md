# TODO App

## Software Requirements

- Nodejs 14.15.1
- NPM 6.14.8
- MongoDB 4.4.5
- Angular CLI 12.0.5

## How to run?

### Server App

- Go inside root > server folder
- npm install
- npm start

### UI App

- Go inside root folder
- npm install
- npm start
- open url `http://localhost:4200/`
